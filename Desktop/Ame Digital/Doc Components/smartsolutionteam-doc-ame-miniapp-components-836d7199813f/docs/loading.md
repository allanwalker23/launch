## Utilização

```xml
<Loading show={show} />
```

## Propriedades

| Propriedade | Descrição                                                                   | Type    | Default | Obrigatório |
|-------------|-----------------------------------------------------------------------------|---------|---------|-------------|
| show        | Essa propriedade recebe true ou false e controla a visibilidade do loading. | Boolean | false   | não         |
