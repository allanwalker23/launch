---

Você pode alterar também o espaço entre as linhas de um texto, utilizando a propriedade **fontHeight**.

### Valores aceitos

- Small: `small` = 100%
- Medium: `medium` = 150%
- Large: `large` = 200%

### Modo de uso

Exemplo:

```xml
<Window>
    <View>
        <Text fontHeight="small"> ... </Text>
        <Text fontHeight="medium"> ... </Text>
        <Text fontHeight="large"> ... </Text>
    </View>
</Window>
```
