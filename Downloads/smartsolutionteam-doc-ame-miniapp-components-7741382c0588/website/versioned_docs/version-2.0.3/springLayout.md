## Utilização 

```xml

head={(

    <Paragraph>Rodapé</Paragraph>

```


## Propriedades

### Head

Elemento que componhe o cabeçalho da tela.

### Body

Corpo de um texto descritivo. 

### footer
Elemento que componhe o rodapé da tela.
