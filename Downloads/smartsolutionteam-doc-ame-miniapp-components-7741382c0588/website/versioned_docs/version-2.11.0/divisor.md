## Utilização

```xml
<Divisor/>
```

## Propriedades

### line
- Type: Boolean
- Opcional

Exibe uma linha contínua.

```xml
<Divisor line />
```

![divisor](assets/images_components/v2.6.0/divisor_2.png)

### light
- Type: Boolean
- Since: 2.8.0
- Opcional

Essa propriedade deixa a linha mais clara

```xml
<Divisor line light />
```
