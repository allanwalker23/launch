Componente que define um bloco com altura e largura definida.

## Utilização

```xml
<Block width={'300px'} heigth={'auto'}>
{...children}
</Block>
```

## Propriedades

### Width

Valor atribuído em string para definir largura do bloco.

### Heigth

Valor atribuído em string para definir altura do bloco.
